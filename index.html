Installing QMK on Sonix Keyboards

Welcome to the SonixQMK Docs website! We made this resource to make installation of QMK a breeze on these fantastic low-cost keyboards. QMK is an open source firmware which gives many great features such as custom layouts, custom RGB patterns and even macros. Our goal for this website is to make it easy to understand installation. If you ever have any problem please join our Discord (link at the top) and ask away in the #firmware channel. Please remember that this project is not owned or operated by any of the supported keyboard manufacturers, Sonix, or the QMK project and this must be done at your own risk. If you brick your board it may require a SWD programmer, these only cost a few dollars. With that out of the way, have fun with SonixQMK!
